Cloudmobility Charts

This is the Helm repo for Cloudmobility charts.
Follow the steps below to start deploying any of them in your cloudmobility namespace. More info at https://github.com/cloudmobility/charts

$ helm repo add cloudmobility https://cloudmobility.github.io/charts
$ helm search repo cloudmobility
$ helm install my-release cloudmobility/chart-name --namespace my-namespace
